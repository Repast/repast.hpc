50 /**
51  * Agent identity information. An Agent ID consists of four values:
52  * 1) a numerical identifier; 2) the process on which the agent
53  * was created; 3) a numerical identifier that indicates the
54  * agent's type (in simulation semantic terms, not a software object
55  * type); and 4) the process on which the agent is a local agent.
56  * Each agent should be uniquely identified by an AgentId using the
57  * first three of the four values, which should be immutable. The
58  * fourth value can change throughout the simulation.
59  */
60 class AgentId {
