50  * Used to build NCDataSets to record data in NetCDF format. Steps for use
51  * are:
52  * <ol>
53  * <li>Create a NCDataSetBuilder.
54  * <li> Add NCDataSources to the builder using the createNCDataSource functions. Each
55  * DataSource defines a variable and where the data for that variable will be retrieved.
56  * Recording data on the NCDataSet produced by the builder will record this data for
57  * each variable.
58  * <li>Call createDataSet to create the NCDataSet.
59  * <li>Schedule calls to record and write on the NCDataSet.
60  * </ol>
61  */

73  * @param file the name of the file to write to. Only rank 0 will
74  * actually write to this file.
75  * @param schedule the schedule to get tick counts from

77  NCDataSetBuilder(std::string file, const Schedule& schedule);
